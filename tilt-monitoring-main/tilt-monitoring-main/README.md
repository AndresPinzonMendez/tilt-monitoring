
# Tilt Monitoring of Electricity Poles

## Set up the development environment

```bash

# Clone the repo 
git clone https://github.com/norbertherbert/tilt-monitoring.git

# Enter the directory
cd tilt-monitoring  

# Install the dependencies
npm install

# Start development server
npm run dev
```

## Build the app

```bash
# Build the app
npm run build
```

## Deploy to GitHub Pages

Push a commit to the `main` branch and GitHub Actions will build the app and publish the generated `dist` folder to GitHub Pages automatically.

To enable this in GitHub:

1. Open the repository `Settings`.
2. Go to `Pages`.
3. Set `Source` to `GitHub Actions`.
